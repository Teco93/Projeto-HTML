Quando surgiu?

(Voltar)

O HTML foi criado em 1991, por Tim Berners-Lee, no CERN (European Council for Nuclear Research) na suíça. Inicialmente o HTML foi projetado para interligar instituições de pesquisa próximas, e compartilhar documentos com facilidade. Em 1992, foi liberada a biblioteca de desenvolvimento WWW ( World Wide Web), uma rede de alcance mundial, que junto com o HTML proporcionou o uso em escala mundial da WEB. O HTML é uma linguagem de marcação. Estas linguagens são constituídas de códigos que delimitam conteúdos específicos, segundo uma sintaxe própria. O HTML tem códigos para criar paginas na web. Estes códigos que definem o tipo de letra, qual o tamanho, cor, espaçamento, e vários outros aspectos do site.

O HTML foi a primeira linguagem de nível mundial, porem não é a única. Existem muitas outras linguagens destinadas á criação de paginas da web, porém o HTML ainda prevalece. Atualmente já é possível integrar varias linguagens na mesma pagina da Web, sendo possível usar duas ou mais linguagens no mesmo site.

Características Principais

(Voltar)

As características principais do HTML são:

  1. Documentos HTML são arquivos escritos em ASCII - texto;
  2. Podem ser criados em qualquer editor de texto ( vi, emacs, edit, notepad);
  3. Existem editores específicos para várias plataformas;
  4. Existem conversores de vários formatos, por exemplo, doc para html; e
  5. A unidade mínima de informaçãoéa“página”.

O que são tags?

(Voltar)

As tags são usadas para informar ao navegador a estrutura do site. Ou seja: quando se escreve um código em HTML, as tags serão interpretadas pelo navegador, produzindo assim a estrutura e o conteúdo visual da página.

A principal característica das tags é estarem sempre dentro dos sinais de chevron (sinal de “maior que” e “menor que”), ou seja: < >.

As tags HTML são divididas em dois tipos: as que precisam de fechamento e as que não precisam de fechamento. As tags que precisam de fechamento possuem a sintaxe , já as que não precisam de fechamento possuem como estrutura .

Além disso, uma mesma tag pode receber um ou mais atributos, que possuirá um valor que modifica sua estrutura ou funcionalidade.

Exemplo de Tags:

  • Tags de Título: (h1)...(h6)
  • Tags de Texto: i, Strong, u, small, sub, sup, font, del, q e p
  • Tags de Citação Curta e Longa: abbr e blockquote
  • Tags de Ordem e lista: ol e ul; li
  • Tag de Ancoragem: a
  • Tag de Elemento: hr, mark

    Tags de Texto:

    Essas tags são utilizadas como estruturas de definição de um texto, sendo fáceis de usar, pois exigem muito pouco do programador.

    Exemplo: Itálico é usado para expressar um texto ou frase em itálico, Strong é usado para formatar o texto e transformá-lo em negrito, 'U' sublinha todo o texto, 'Small' reduz o tamanho do texto, 'Sub' coloca as palavras abaixo da linha de escrita, 'Sup' é usado para elevar os textos ou destacar as palavras, 'Del' é utilizado quando se quer mostrar que uma palavra foi abolida do texto, uma frase ou um texto todo, 'Q' destaca uma frase ou um texto com aspas duplas, 'Font' é usufruida com o intuito de formatar os textos que se quer(usei a cor verde) e todas essas palavras incluídas na tag(P).

    (Voltar)

    Tags de Citação (curta e longa):

    Os marcadores de citação são usados com frequência para destacar textos - grandes ou pequenos: "Blackquote" e "q"

    Exemplo:

    A 'Blockquote' é utilizada para destacar citações longas, mudando o alinhamento para torná-lo igual aos outros.

    A tag 'q' é utilizada para pequenas citações.

    (Voltar)

    Tags de Ordem e Lista:

    As tags de ordem são utilizadas para oferecer ao programador a organização dos itens que o mesmo deseja encorpar em seu código, sendo que ela pode ser ordenada (OL) e desordenada (UL). A lista de itens (LI) é usada para corroborar com as tags UL e OL.

    Exemplo: Ordenada

    1. gema
    2. clara
    3. casca de ovo

    Exemplo: Desordenada

    • bezerro
    • vaca
    • boi

    Tags de Elementos:

    As tags 'Mark' e 'Hr' são elementos diferetes, porém, simples: O primeiro busca destacar qualquer trecho de um texto e o segundo faz uma quebra temática entre os elementos de paragráfos.
